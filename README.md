# Coin Flip Simulator

A simple JavaScript program that simulates flipping 3 coins 6 times and displays the results in a visual format with statistics.

## Features

- **Coin Flip Simulation**: Flip 3 coins 6 times with a single button click
- **Visual Display**: Shows results with color-coded coins (Green = Heads, Orange = Tails)
- **Statistics Chart**: Bar chart comparing total Heads vs Tails counts
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Open `index.html` in your web browser
2. Click the "Flip 3 Coins (6 Times)" button
3. View the results displayed vertically (Flip 6 at top, Flip 1 at bottom)
4. Check the statistics bar chart below

## Files

- `index.html` - Main HTML file
- `style.css` - Styling and layout
- `script.js` - JavaScript logic for coin flipping and chart generation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js library
